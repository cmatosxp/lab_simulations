Shell operators implemented in this lab

Operators change how commands are connected. These examples behave inside the simulator instead of being display-only documentation.

Symbol / Operator Description
; Run commands sequentially: pwd; ls.
&& Run the next command only when the previous command succeeds: cd Documents && ls.
|| Run the next command only when the previous command fails: cd missing || pwd.
| Pipe output into grep: cat welcome.txt | grep Welcome.
> Write command output to a file, replacing existing content: echo "hello" > note.txt.
>> Append command output without replacing existing content: echo "again" >> note.txt.
